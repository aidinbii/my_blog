1. Алгебра сабагы, 8-класс: Көп мүчөлөр менен амалдар

Тема: Көп мүчөлөрдү кошуу, кемитүү. Бир мүчөнү көп мүчөгө көбөйтүү. Көп мүчөнү көп мүчөгө көбөйтүү. Жалпы көбөйтүүчүнү кашаанын сыртына чыгаруу.

Максаты: Көп мүчөлөр менен негизги амалдарды аткарууну үйрөнүү.

1. Көп мүчөлөрдү кошуу жана кемитүү

Аныктама:

  • Көп мүчө - бул кошуу жана кемитүү белгилери менен байланышкан бир мүчөлөрдөн турган алгебралык туюнтма.
  • Бир мүчө - сандардын, өзгөрмөлөрдүн жана алардын даражаларынын көбөйтүндүсү.

Эреже: Көп мүчөлөрдү кошуу же кемитүү үчүн:

  1. Кашааларды ачып, алардын алдындагы белгилерди эске алуу керек.
  2. Окшош мүчөлөрдү жыйноо керек.

Мисалдар:

  • (a + 3b - c) + (2a - b + 2c) = a + 3b - c + 2a - b + 2c = (a + 2a) + (3b - b) + (-c + 2c) = 3a + 2b + c
  • (5x² - 2xy + y²) - (3x² + xy - 2y²) = 5x² - 2xy + y² - 3x² - xy + 2y² = (5x² - 3x²) + (-2xy - xy) + (y² + 2y²) = 2x² - 3xy + 3y²

Тапшырмалар:

  1. (4m - 3n + 2) + (m + 5n - 1)
  2. (2x² + 3x - 5) - (x² - x + 2)
  3. (a³ - 2a² + a) + (3a³ + a² - 4a)
  4. (7p²q - 3pq² + q³) - (2p²q + 5pq² - 3q³)

2. Бир мүчөнү көп мүчөгө көбөйтүү

Эреже: Бир мүчөнү көп мүчөгө көбөйтүү үчүн:

  1. Бул бир мүчөнү көп мүчөнүн ар бир мүчөсүнө көбөйтүү керек.
  2. Алынган көбөйтүндүлөрдү кошуу керек.

Мисалдар:

  • 2a (3a² - 4ab + b²) = 2a * 3a² + 2a * (-4ab) + 2a * b² = 6a³ - 8a²b + 2ab²
  • -5xy (x²y - 2xy² + 3y³) = -5xy * x²y - 5xy * (-2xy²) - 5xy * 3y³ = -5x³y² + 10x²y³ - 15xy⁴

Тапшырмалар:

  1. 3x (x² - 2x + 4)
  2. -2ab (a²b - 3ab² + 4b³)
  3. 4m²n (m³n² - 2m²n³ + 5mn⁴)
  4. -pq (p²q² - 3pq + 5)

3. Көп мүчөнү көп мүчөгө көбөйтүү

Эреже: Көп мүчөнү көп мүчөгө көбөйтүү үчүн:

  1. Биринчи көп мүчөнүн ар бир мүчөсүн экинчи көп мүчөнүн ар бир мүчөсүнө көбөйтүү керек.
  2. Алынган көбөйтүндүлөрдү кошуу керек.

Мисалдар:

  • (a + 2)(b - 3) = a * b + a * (-3) + 2 * b + 2 * (-3) = ab - 3a + 2b - 6
  • (x - 2)(x² + 3x - 1) = x * x² + x * 3x + x * (-1) - 2 * x² - 2 * 3x - 2 * (-1) = x³ + 3x² - x - 2x² - 6x + 2 = x³ + x² - 7x + 2

Тапшырмалар:

  1. (m + 3)(n - 5)
  2. (2x - 1)(x + 4)
  3. (a - 2)(a² + 2a + 4)
  4. (y + 3)(y² - 3y + 9)

4. Жалпы көбөйтүүчүнү кашаанын сыртына чыгаруу

Эреже: Жалпы көбөйтүүчүнү кашаанын сыртына чыгаруу үчүн:

  1. Көп мүчөнүн бардык мүчөлөрүнүн жалпы көбөйтүүчүсүн табуу керек.
  2. Көп мүчөнүн ар бир мүчөсүн бул көбөйтүүчүгө бөлүү керек.
  3. Алынган көп мүчөнү кашаанын ичине жазып, жалпы көбөйтүүчүнү кашаанын алдына жазуу керек.

Мисалдар:

  • 6a²b - 3ab² = 3ab (2a - b)
  • 12x³y² - 8x²y³ + 4xy⁴ = 4xy² (3x² - 2xy + y²)

Тапшырмалар:

  1. 10m²n - 5mn²
  2. 14a³b² + 21a²b³ - 7ab⁴
  3. 8p⁴q² - 12p³q³ + 4p²q⁴
  4. 15x⁵y³ - 20x⁴y⁴ + 5x³y⁵

Кошумча тапшырмалар:

  1. (2a - 3b + c)(a + b - 2c)
  2. (x² - 2x + 1)(x² + 2x + 1)
  3. 18a⁴b³ - 12a³b⁴ + 6a²b⁵ - 30a⁵b²
  4. (m + n)(m² - mn + n²)

Кеңештер:

  • Кашааларды ачканда белгилерге көңүл буруңуз.
  • Окшош мүчөлөрдү жыйноону унутпаңыз.
  • Жоопторуңузду өзгөрмөлөрдүн маанилерин коюп текшериңиз.
